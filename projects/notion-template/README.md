# 📔 Notion Template - 2026 年度計劃系統

## 產品描述
一個完整既 Notion 模板，幫助用戶管理全年既目標、習慣、項目同財務。

## 包含內容

### 1. 年度目標 Dashboard
- 12個月目標追蹤
- 季度回顧系統
- 成功指標 (KPIs)
- 視覺化進度圖表

### 2. 習慣追蹤系統
- 每日/每週習慣清單
- 連勝記錄 (Streak)
- 獎勵機制
- 月度習慣報告

### 3. 項目管理
- 優先級分類 (Eisenhower Matrix)
- 截止日期追蹤
- 進度百分比
- 依賴關係

### 4. 財務追蹤
- 收入/支出記錄
- 預算分類
- 儲蓄目標
- 月度財務報告

### 5. 閱讀清單
- 書籍追蹤
- 筆記整理
- 評論系統

### 6. 學習日誌
- 技能學習追蹤
- 課程記錄
- 進度評估

## 特色
